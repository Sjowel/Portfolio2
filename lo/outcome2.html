Reflection

How did it go and do you notice any difficulties and challenges?


I didn't really have difficulties with this learning outcome so it went pretty well. I know how to use version control on GitHub and how to commit my changes of any project or my portfolio. The only difficult part was when we worked together on the development project, I couldn't commit. Because we worked together in a Live Share server on VS Code only one person could commit.
The coding part wasn't that hard, it is nice to have a design you can make which is already made in Figma, and Live Server worked really good to work together, you don't really have any conflicts when you commit your changes or they can be fixed immediately.

Do you want to grow in this learning outcome?


Yes, of course I want to grow in this learning outcome. I want to learn more about JavaScript because my knowledge about it isn't that big yet. I also want to learn more about version control because I think it is a good way to keep track of your changes and improvements.
