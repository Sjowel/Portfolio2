Version control

What did I do?

I set up a repository in GithHub.

Why did I do that?

I did this so me and my group can work together on the same project.

How did I do it?

I went to GitHub and made a new repository. I added my group members to the repository so we can all work together on the project. I added this repository to my VSCode.

What were the results?

A group member of mine suggested using Live server so we can work live together and everything is updated in real time which makes it easier to work together and avoid conflicts.

Conclusion

I liked working with Live server because if something went wrong you can go back to your previous commit and fix it immediately which makes it much faster than finding out what went wrong and fix it in the repository itself.
