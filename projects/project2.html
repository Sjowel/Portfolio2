What is the project about?

For Fix that UX we had to fix the UX of a premium calculator on an health insurance website. We had to choose between CZ, Zilveren Kruis or ZEKUR.

Research Zilveren Kruis

What did I do?

I did research on the premium calculator on the website of Zilveren Kruis

Why did I do that?

I did this to find out which calculator had the most pain points so me and my group could improve that premium calculator.

How did I do it?

Me and Angelina made a step-by-step plan of how you can go through the premium calculator on the website of Zilveren Kruis. We wrote down all the steps and all the points we encountered while we went through the process. I also did 2 usertests (1 younger person(between 18 and 35) and one older person (35+)) on this website to look if we missed a pain point or if it was easy enough to go through the website.

What were the results?

Zilveren Kruis was the websit with the least pain points so we didn't choose this one to improve because we wanted a little bit of a challenge. We chose the website of ZEKUR because this was the website with the most pain points

Conclusion

If you are not sure what website to improve, do research on all of them and look which one has the most pain points. This way you can improve the website with the most pain points and you can learn the most from it.

Usertests

Usertests Zilveren Kruis

What did I do?

I did 2 usertests on the premium calculator of Zilveren Kruis.

Why did I do that?

I did this to see if we missed any pain points in our research and to see if the website was easy enough to go through.

How did I do it?

I set up 2 usertests, one with a younger person (between 18 and 35) and one with an older person (35+). I let them go through the premium calculator and asked them questions about the website and if they had any pain points.

What were the results?

The results were that Zilveren Kruis was the website with the least pain points so we didn't choose this one to improve. The pain points I got out of the usertests were the following:
- The user went to the calculator with a detour.
- The user thought he had to fill in all the tabs, but some were optional.
- The user didn't like it that at the supplementary insurance there were three next to each other and two underneath it which made it harder to compare them.
- The user thought the letters on the website were a little too small.
- The user thought the picture in the beginning of the lady wasn't relevant to the website.
Improvements I think you can make based on the feedback:
- Make the calculator easier to find. For example a clear button on the home page.
- Make it clear which tabs are optional and which are not.
- Make the supplementary insurance easier to compare.
- Make an option where you can increase or decrease the size of the letters.

Based on all the pain points we had gathered from the Zilveren Kruis website, it were the least so we didn't choose this website to improve.

Usertest ZEKUR

What did I do?

I did one usertest on the ZEKUR website.

Why did I do that?

I did this because as a group we choose to improve the ZEKUR website because this one had the most pain points. I wanted to find out if we missed any pain points on this website.

How did I do it?

I used the same usertest as I did with the Zilveren Kruis website. I let the user go through the website and asked him questions about the website and if he had any pain points.

What were the results?

The pain points I got out of the usertest were the following:
- The user had to look up what some things meant because it wasn't clearly explained on the website.
- The used pressed the wrong button to calculate the premium.(this one on the home page was for ZZP)
- The user thought you had to fill in everything while some things were just optional.
- The user didn't like it that you had to scroll down to see how much the insurance would cost.
Improvements I think you can make based on the feedback:
- Make it clear what everything means on the website.
- Make it clear which tabs are optional and which are not.
- Make the premium calculator easier to find.
- Make it clear how much the insurance would cost without scrolling down.

These were just a few pain points I found because of my usertest. But because everyone else did some there were way more.
